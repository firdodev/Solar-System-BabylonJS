## Getting Started

### Prerequisites

First we need to install all the modules
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation
1. Clone the repo
   ```sh
   git clone https://github.com/firdodev/Solar-System-BabylonJS.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start npm server
   ```js
   npm start
   ```
